# Yii

Yii is a high-performance, package-based PHP framework for developing modern applications.  It provides a full set of proven, ready-to-use features including caching, logging, a template engine, data abstraction, validation, authentication and authorization, routing, a middleware pipeline, a dependency injection container, database access via query builder and Active Record, migrations, event handling, and console tools. It's especially suitable for developing large-scale applications such as portals, content management systems, e-commerce, REST APIs, etc.
