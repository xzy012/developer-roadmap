# Good Layout Rules

Good layout rules are the visual design principles that guide how elements are arranged on a screen, covering alignment, spacing, hierarchy, and grouping. Following them makes an interface easier to scan and reduces the mental effort needed to find information. These rules apply regardless of which prototyping tool is used to build the design.
