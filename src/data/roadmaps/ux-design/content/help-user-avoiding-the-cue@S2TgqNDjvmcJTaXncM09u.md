# Help user Avoiding the Cue

Helping a user avoid the cue means removing or hiding the trigger that sets off an unwanted habit. If a notification prompts compulsive checking, muting or removing that notification eliminates the starting point of the loop. This is often the simplest way to break a habit, since a habit cannot start without its cue.
