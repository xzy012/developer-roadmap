# Defaulting

Defaulting means setting a pre-selected option so the user has to opt out rather than opt in. Because most people stick with whatever is already selected, this single design choice can dramatically shift outcomes, such as organ donation rates or retirement plan enrollment. It works because taking no action is the path of least resistance for most users.
