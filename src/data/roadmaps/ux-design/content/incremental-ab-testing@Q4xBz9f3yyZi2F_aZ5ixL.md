# Incremental A/B Testing

Incremental A/B testing compares two versions of a design that differ by a single, small change, showing each version to a separate group of users and measuring which performs better. Because only one variable changes at a time, teams can clearly attribute any difference in outcome to that specific change. It is commonly used to validate individual design decisions before rolling them out to all users.
