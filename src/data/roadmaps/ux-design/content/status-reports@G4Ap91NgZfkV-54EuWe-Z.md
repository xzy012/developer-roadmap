# Status Reports

Status reports are UI elements that summarize the current state of a process, account, or system for the user, such as a dashboard showing recent activity or an order tracking screen. They let users quickly check where things stand without digging through details. This pattern suits moments of limited attention because it delivers a condensed, scannable summary.
