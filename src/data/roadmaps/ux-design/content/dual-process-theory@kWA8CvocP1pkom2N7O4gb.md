# Dual Process Theory

Dual process theory describes two systems of thinking: a fast, automatic, intuitive system and a slower, effortful, analytical system. Most daily decisions rely on the fast system, which is why habits and defaults are so powerful in shaping behavior. Designers use this theory to decide whether an interface should support quick, intuitive action or slow down the user for careful evaluation.
