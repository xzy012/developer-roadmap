# UX Best Practices

UX best practices are established design guidelines proven to improve how users notice, react to, evaluate, and act within a product. They cover getting a user's attention, producing a positive first impression, supporting a favorable conscious evaluation, creating urgency, and making the target action easy. These practices are drawn from repeated observation of what works across many products rather than from a single framework.
