# CREATE Action Funnel
 
The CREATE Action Funnel breaks the path to a user action into six stages: Cue, Reaction, Evaluation, Ability, Timing, and Execution. A user must pass through each stage for the target behavior to happen, and a failure at any stage stops the action. It gives designers a step-by-step way to audit where users drop off before completing a task.
