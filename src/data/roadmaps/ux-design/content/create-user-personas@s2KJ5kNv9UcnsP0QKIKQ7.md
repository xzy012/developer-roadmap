# Create User Personas

Creating user personas means building fictional but research-based profiles that represent the key user types a product serves. Each persona typically includes goals, frustrations, behaviors, and context drawn from interviews, surveys, or analytics rather than guesswork. Personas give a team a shared reference point when making design decisions instead of relying on personal assumptions.
