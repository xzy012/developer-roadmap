# Simple Flowchart

A simple flowchart is a diagram that maps out the steps and decision points in a process using basic shapes like boxes and arrows. It shows the logical path a user or system takes from a starting point to an outcome, including any branches. Teams use it during conceptual design because it is quick to draw and easy for anyone to read.
