# User Flows

A user flow is a diagram showing the path a user takes through a product to complete a specific goal, including the screens and decisions along the way. It focuses on the sequence and logic of an experience rather than visual details. Designers use user flows early in the design process to validate that a proposed structure makes sense before building detailed screens.
