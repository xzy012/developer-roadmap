# Priorization Frameworks

Prioritization frameworks are structured methods for ranking competing ideas or features based on defined criteria, rather than relying purely on opinion. Common examples include Kano, RICE, and MoSCoW, each weighing factors like effort, impact, or user satisfaction differently. Using a shared framework helps a team make prioritization decisions more transparent and easier to explain to stakeholders.
