# Kano

The Kano model classifies features into categories based on how they affect user satisfaction, including basic expectations, performance features that scale with satisfaction, and delighters that create disproportionate happiness when present. It helps teams distinguish between features users expect by default and those that would genuinely excite them. This distinction guides decisions about where extra investment is likely to pay off versus where it is simply required to meet a baseline.
