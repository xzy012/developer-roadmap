# Problem Statements

A problem statement is a concise, specific description of the issue a design effort is meant to solve, usually framed around a particular user and their need. It keeps a project scoped and focused, preventing the team from drifting into unrelated ideas. A well-written problem statement avoids suggesting a specific solution, leaving room for multiple approaches during ideation.
